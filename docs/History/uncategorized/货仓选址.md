**思想**：

* 贪心。
  * 设仓库选址最佳处为 $P$，此时在该位置左侧存在 $m$ 个货仓，右侧存在 $n$ 个货仓，总距离为 $L$。
  * 若更改货仓位置为 $P-1$，则总长度变为 $L - m + n$。
  * 若更改货仓位置为 $P + 1$，则总长度变为 $L + m - n$。
  * 易知位置 $P$ 处有： $L - m + n = L + m - n$ 必然成立，可推出 $m = n$。
  * 综上，最佳位置 $P$ 应使得左右位置的货仓数量相等。

**代码**：
```cpp
#include &lt;bits/stdc++.h&gt;
using namespace std;

const int N = 1e6 + 3;

int a[N];

typedef long long LL;

void solve(){
    int n; cin >> n;
    for(int i = 0; i &lt; n; i ++) cin &gt;> a[i];
    sort(a, a + n);  // 按照位置先后排序
    LL sum = 0;
    for(int i = 0; i &lt; n; i ++) sum += abs(a[n &gt;> 1] - a[i]);  // a[n >> 1]：n 为偶数时 n / 2 和 n / 2 + 1 均可
    cout &lt;&lt; sum &lt;&lt; endl;                                    // n 为奇数时，n / 2 即为所得
}                                                         // 注意，由于数组下标从 0 开始，n &gt;> 1 后无需做偏移处理
int main(){
    solve();
    return 0;
}
```
