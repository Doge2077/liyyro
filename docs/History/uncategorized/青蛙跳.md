**思想** ：

  * 思维。
  * 青蛙一共跳了 `k` 次，则： 
```java
* 当 `k` 为奇数时，向右边跳了 `k / 2 + 1` 次，向左边跳了 `k / 2` 次。
* 当 `k` 为偶数时，向右边跳了 `k / 2` 次，向左边跳了 `k / 2` 次。
```



**代码** ：
```java


#include &lt;bits/stdc++.h&gt;
using namespace std;

typedef long long LL;

void solve(){
    LL a, b, k; cin >> a >> b >> k;
    cout << a * (k / 2 + (k % 2LL != 0)) - b * (k / 2) << endl;
}

int main(){
    int _; cin >> _;
    while(_ --) solve();
    return 0;
}
```
