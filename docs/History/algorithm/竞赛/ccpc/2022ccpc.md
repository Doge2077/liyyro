## A. Mocha 上小学啦

---

[原题链接](https://codeforces.com/gym/103941/attachments)

**题意**：

* 一个数由互不相同的 $n$ 位数字组成。
  * 输出 $n$ 位的满足上述条件的最小整数（不含前导零）。
  * 不存在则输出 $-1$。

---

**思路**：

* 签到题。
  * 当 `n &lt;= 10` 时，最小的 $n$ 位数的序列为 `[1, 0, 2, 3, 4, 5, 6, 7, 8, 9]`。
  * 当 `n &gt; 10` 时不存在可以满足条件的数。

---

**代码**：
```cpp
#include &lt;iostream&gt;
#include &lt;cstring&gt;
#include &lt;cstdio&gt;
#include &lt;algorithm&gt;
#include &lt;cmath&gt;
#include &lt;sstream&gt;
#include &lt;vector&gt;
#include &lt;queue&gt;
#include &lt;stack&gt;
#include &lt;map&gt;
#include &lt;set&gt;
#include &lt;unordered_map&gt;
#include &lt;unordered_set&gt;

using namespace std;

#define IOS ios::sync_with_stdio(false),cin.tie(nullptr),cout.tie(nullptr)
#define re register
#define fi first
#define se second
#define endl '\n'

typedef long long LL;
typedef pair&lt;int, int&gt; PII;
typedef pair&lt;LL, LL&gt; PLL;

const int N = 1e6 + 3;
const int INF = 0x3f3f3f3f, mod = 1e9 + 7;

    int n; cin >> n;

    else {
        for (int i = 0; i &lt; n; i++) cout &lt;&lt; a[i]; // 注意交换1和0的顺序
    }
    return;
}

int main() {
    IOS;
    int _ = 1;
    // cin &gt;> _;
    while (_--) {
        solve();
    }
    return 0;
}
```

## E. Serval 的俳句

---

[原题链接](https://codeforces.com/gym/103941/attachments)

**题意**：

给定一个长度为 $n$ 的字符串 $S$。
  求是否存在长度为 $17$ 的子序列满足：
  * $S'_1, S'_2, S'_3, S'_4, S'_5$ 为同一个字符；
  * $S'_6, S'_7, \dots, S'_{11}, S'_{12}$ 为同一个字符；
  * $S'_{13}, S'_{14}, S'_{15}, S'_{16}, S'_{17}$ 为同一个字符。

存在输出该子序列，不存在则输出 `none`。

---

**思路**：

签到题。
  * 暴力枚举 $S_i$，统计满足的 $S'_1 - S'_5$，若可以找到满足的子序列，标记该字符后进入下一层循环；
  * 暴力枚举 $S_j$，统计满足的 $S'_6 - S'_{12}$，若可以找到满足的子序列，标记该字符后进入下一层循环；
  * 暴力枚举 $S_k$，统计满足的 $S'_{13} - S'_{17}$，若可以找到满足的子序列，则将上面标记的直接输出即可。
  * 当 `n &lt; 17` 时或者没有找到满足条件的子序列，则输出 `none`。
  * 由于每一段的子序列很短，最坏的情况是 $26^3 \times 5^2 \times 7 = 3075800$，实际上的情况会远小于该时间复杂度，所以完全 OK。

---

**代码**：
```cpp
#include &lt;iostream&gt;
#include &lt;cstring&gt;
#include &lt;cstdio&gt;
#include &lt;algorithm&gt;
#include &lt;cmath&gt;
#include &lt;sstream&gt;
#include &lt;vector&gt;
#include &lt;queue&gt;
#include &lt;stack&gt;
#include &lt;map&gt;
#include &lt;set&gt;
#include &lt;unordered_map&gt;
#include &lt;unordered_set&gt;

using namespace std;

#define IOS ios::sync_with_stdio(false),cin.tie(nullptr),cout.tie(nullptr)
#define re register
#define fi first
#define se second
#define endl '\n'

typedef long long LL;
typedef pair&lt;int, int&gt; PII;
typedef pair&lt;LL, LL&gt; PLL;

const int N = 200;
const int INF = 0x3f3f3f3f, mod = 1e9 + 7;
const double eps = 1e-6, PI = acos(-1);

void solve() {
    int n; cin >> n;
    if (n &lt; 17) { // 长度不够，不可能找到
        cout &lt;&lt; "none" &lt;&lt; endl;
        return;
    }
    string s; cin &gt;> s;
    int vis1[N] = {0}; // 统计相同的前5个字符
    for (int i = 0; i &lt; s.size(); i++) {
        vis1[s[i]]++;
        if (vis1[s[i]] == 5) {
            char op1 = s[i]; // 标记前5个字符
            int vis2[N] = {0}; // 统计相同的中间7个字符
            for (int j = i + 1; j &lt; s.size(); j++) {
                vis2[s[j]]++;
                if (vis2[s[j]] == 7) {
                    char op2 = s[j]; // 标记中间7个字符
                    int vis3[N] = {0}; // 统计相同的后5个字符
                    for (int k = j + 1; k &lt; s.size(); k++) {
                        vis3[s[k]]++;
                        if (vis3[s[k]] == 5) { // 找到最后5个字符，直接输出
                            for (int u = 0; u &lt; 5; u++) cout &lt;&lt; op1;
                            for (int u = 0; u &lt; 7; u++) cout &lt;&lt; op2;
                            for (int u = 0; u &lt; 5; u++) cout &lt;&lt; s[k];
                            return;
                        }
                    }
                }
            }
        }
    }
    // 找不到满足条件的序列
    cout &lt;&lt; "none" &lt;&lt; endl;
}

int main() {
    IOS;
    int _ = 1;
    // cin &gt;> _;
    while (_--) {
        solve();
    }
    return 0;
}
```

## F. 集合之和

---

[原题链接](https://codeforces.com/gym/103941/attachments)

**题意**：

* 对于两个有限的数集 $A, B$，定义 $A + B$ 为：
  $A + B = \{x + y \mid x \in A, y \in B\}$
* 记有限数集 $A$ 的元素个数为 $|A|$。
* 给定 $n$，求满足 $|A + A| = n$，且满足 $\forall x \in A, 0 \le x \le 5 \times 10^5$ 的数集 $A$。
* 若存在 $A$ 满足上述条件，输出集合 $A$ 的大小并输出其元素（任意解均可），不存在则输出 $-1$。

---

**思路**：

* 思维，规律，构造。

构造简单数集 $A = \{0,1,2,\dots,k\}, |A| = k$，则 $A + A = \{0,1,2,\dots,2k\}, |A+A| = 2k + 1$。

通过上述构造，显然当 $n$ 为奇数时一定有解，我们只需要构造满足如上规则的数集即可。

接下来讨论 $n$ 为偶数的情况：
* $|A| = 2, |A+A| = 3$，当 $n = 2$ 时无解；
* $|A| = 3, 5 \le |A+A| \le 6$，当 $n = 4$ 时无解；
* 其他情况下，构造数集 $A = \{0,2,3,\dots,k\}$，则 $A+A = \{0,2,3,\dots,2k\}, |A+A| = 2k$。

综上所述，根据 $n$ 的奇偶性按照上述规则构造即可。

---

**代码**：
```cpp
#include &lt;iostream&gt;
#include &lt;cstring&gt;
#include &lt;cstdio&gt;
#include &lt;algorithm&gt;
#include &lt;cmath&gt;
#include &lt;sstream&gt;
#include &lt;vector&gt;
#include &lt;queue&gt;
#include &lt;stack&gt;
#include &lt;map&gt;
#include &lt;set&gt;
#include &lt;unordered_map&gt;
#include &lt;unordered_set&gt;

using namespace std;

#define IOS ios::sync_with_stdio(false),cin.tie(nullptr),cout.tie(nullptr)
#define re register
#define fi first
#define se second
#define endl '\n'

typedef long long LL;
typedef pair&lt;int, string&gt; PII;
typedef pair&lt;LL, LL&gt; PLL;

const int N = 1e6 + 3;
const int INF = 0x3f3f3f3f, mod = 1e9 + 7;
const double eps = 1e-6, PI = acos(-1);

int n;

void solve() {
    cin >> n;
    if (n == 2 || n == 4) {
        cout &lt;&lt; -1 &lt;&lt; endl;
        return;
    }
    if (n % 2 != 0) {  // 奇数构造 0,1,2,...,(n-1)/2
        cout &lt;&lt; (n - 1) / 2 + 1 &lt;&lt; endl;
        for (int i = 0; i &lt;= (n - 1) / 2; i++) cout &lt;&lt; i &lt;&lt; ' ';
    } else {  // 偶数构造 0,2,3,...,n/2
        cout &lt;&lt; n / 2 &lt;&lt; endl;
        cout &lt;&lt; 0 &lt;&lt; ' ';
        for (int i = 2; i &lt;= n / 2; i++) cout &lt;&lt; i &lt;&lt; ' ';
    }
}

int main() {
    IOS;
    int _ = 1;
    while (_--) {
        solve();
    }
    return 0;
}
```

## G. Mocha 上大班啦

[原题链接](https://codeforces.com/gym/103941/attachments)

**题意**：

* 给定 $n$ 个 $01$ 串以及 $q$ 次操作，每次操作会把第 $i$ 个串的 $l-r$ 位替换为和第 $j$ 个串的 $l-r$ 位分别进行与运算的结果。
  * 第 $i$ 次操作有概率 $p_i$ 成功。
  * 询问最后 $n$ 个串与运算得到的串中 $1$ 的个数。

---

**思路**：

* 思维题。
  * 概率期望套皮，其实无需处理。
  * 只要一位中出现过 $0$，由于进行的是“与”位运算，操作多少次都不会影响任何一个 $01$ 串第 $i$ 位的值，结果一定是 $0$。
  * 故直接计算初始 $01$ 串与操作后的 $1$ 的个数即为答案。

---

**代码**：
```cpp
#include &lt;iostream&gt;
#include &lt;cstring&gt;
#include &lt;cstdio&gt;
#include &lt;algorithm&gt;
#include &lt;cmath&gt;
#include &lt;sstream&gt;
#include &lt;vector&gt;
#include &lt;queue&gt;
#include &lt;stack&gt;
#include &lt;map&gt;
#include &lt;set&gt;
#include &lt;unordered_map&gt;
#include &lt;unordered_set&gt;

using namespace std;

#define IOS ios::sync_with_stdio(false),cin.tie(nullptr),cout.tie(nullptr)
#define re register
#define fi first
#define se second
#define endl '\n'

typedef long long LL;
typedef pair&lt;int, string&gt; PII;
typedef pair&lt;LL, LL&gt; PLL;

const int INF = 0x3f3f3f3f, mod = 1e9 + 7;
const double eps = 1e-6, PI = acos(-1);

const int N = 1010;

char a[N][4 * N];

int n, m;

void solve() {
    int res = 0;
    cin >> n >> m;
    for (int i = 0; i &lt; n; i++) cin &gt;> a[i];

    int q; cin >> q;
    while (q--) {
        for (int i = 0; i &lt; 5; i++) {
            int op; cin &gt;> op;
        }
    }

    for (int i = 0; i &lt; m; i++) {
        int cnt = 1;
        for (int j = 0; j &lt; n; j++) {
            if (a[j][i] == '0') cnt = 0;
        }
        res += cnt;
    }

    cout &lt;&lt; res &lt;&lt; endl;
}

int main() {
    IOS;
    int _ = 1;
    while (_--) {
        solve();
    }
    return 0;
```

## H. 旋转水管

[原题链接](https://codeforces.com/gym/103941/attachments)

**题意**：

给定一个 \(2 \times m\) 的水管图，水管有 `I`、`L` 两种类型，且两种水管均可以旋转。问是否可以通过旋转水管，使得水从第 \(x\) 列流入后从第 \(y\) 列流出。

**思路**：

采用 **DFS** 搜索。

- 水管的出水状态与流入状态相关，因此需要记录上一个格子流入下一个格子时的方向状态。
- 分析状态转换规则可知：
    - 对于 `I` 形的水管，流出状态与流入状态相同。
    - 对于 `L` 形的水管：
        - 当流入方向是向上 (\(\uparrow\)) 或向下 (\(\downarrow\)) 时，流出方向有两种可能：向左 (\(\leftarrow\)) 或向右 (\(\rightarrow\))。
        - 当流入方向是向左 (\(\leftarrow\)) 或向右 (\(\rightarrow\)) 时，流出方向有两种可能：向上 (\(\uparrow\)) 或向下 (\(\downarrow\))。

根据上述分析，我们可以定义对应方向状态的坐标偏移量，在搜索过程中通过加上该偏移量来实现坐标转移，从而判断水的流动是否合法。

**代码**：

```cpp
#include &lt;iostream&gt;
#include &lt;cstring&gt;
using namespace std;

const int MAXM = 1000;
int m, sl, el;
int flag;
int vis[2][MAXM];
char mp[2][MAXM];

void dfs(int l, int r, int st) {
    if (flag == 1) return;  //满足条件直接返回

    if (l == 1 && r == el - 1) {  //到达出口的情况合法
        flag = 1;  //标记答案
        return;
    }

    if (l &lt; 0 || l &gt;= 2 || r &lt; 0 || r &gt;= m) return;  //越界直接返回

    if (vis[l][r]) return;  //走过该点返回

    vis[l][r] = 1;  //标记走过

    if (l >= 0 && l &lt; 2 && r &gt;= 0 && r &lt; m) {
        if (mp[l][r] == 'I') {  //进入状态和流出状态相同
            if (st == 0) dfs(l + 1, r, 0);
            if (st == 1) dfs(l, r + 1, 1);
            if (st == 2) dfs(l - 1, r, 2);
            if (st == 3) dfs(l, r - 1, 3);
        } else {
            if (st == 0 || st == 2) {  //从下面或上面流入的状态
                dfs(l, r + 1, 1);
                dfs(l, r - 1, 3);
            }
            if (st == 1 || st == 3) {  //从左边或右边流入的状态
                dfs(l + 1, r, 0);
                dfs(l - 1, r, 2);
            }
        }
    }

    vis[l][r] = 0;  //回溯时恢复现场
}

bool solve() {
    cin &gt;> m >> sl >> el;
    for (int i = 0; i &lt; 2; i++) cin &gt;> mp[i];

    flag = 0;  //多实例，初始化
    for (int i = 0; i &lt; 2; i++) {
        for (int j = 0; j &lt;= m; j++) vis[i][j] = 0;
    }

    dfs(0, sl - 1, 0);

    return flag;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int _ = 1;
    cin &gt;> _;

    while (_--) {
        if (solve()) cout << "YES" << endl;
        else cout << "NO" << endl;
    }

    return 0;
}
```
