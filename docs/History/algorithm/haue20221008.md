**题目大意** ：

* 给定矩形范围的左上角和右下角的坐标。
  * 判断若干点到该矩形边界的最短距离，输出最短距离的点的编号。

---

**思想** ：

* 思维题。
  * 对于一个点，可以暴力枚举其到四条边的直线距离和到四个顶点的直线距离，取最小值即可。

## F. HF波那契数列

---

[原题链接](http://www.haueacm.top/problem.php?cid=1003&pid=5)

**题目大意**：

*   类似斐波那契数列，构造 \(HF\) 波那契数列：
    $$
    G_n = G_{n - 1} + G_{n - 2}
    $$
    其中 \(G_0 = 1, G_1 = t (0 &lt; t)\)。
*   给定 \(i\) 和 \(G_i\)，求满足条件的 \(G_i \mod 19960515\) 的值。
*   若不存在，则输出 \(-1\)。

---

**思想**：

思维题：
1.  列出部分 $G$ 波那契数列：
    *   $G_0, G_1, G_2, G_3, G_4, G_5 \dots$
    *   $1, t, t+1, 2t+1, 3t+2, 5t+3 \dots$
2.  设 $G_i = x_i \cdot t + y_i$，其中 $x_i$ 和 $y_i$ 为系数。从 $i=0$ 开始观察可知：
    *   $x_i$ 满足：$0, 1, 1, 2, 3, 5 \dots$，即 $x$ 从 $i=1$ 开始构成斐波那契数列。
    *   $y_i$ 满足：$0, 0, 1, 1, 2, 3 \dots$，即 $y$ 从 $i=2$ 开始构成斐波那契数列。
3.  因此，对于给定的 $G_i$ 值 $g$，我们可以由 $t = (g - y_i) / x_i$ 求得对应的 $t$ 值（需满足 $t&gt;0$ 且整除）。
4.  由于数据范围可能很大，但斐波那契数列增长极快，我们只需预处理出前几十项（例如 90 项以内，因为之后数值将超出常用整数范围）的原始斐波那契数列，即可直接查询 $x_i$ 和 $y_i$，而无需递推。
5.  注意：最终输出结果需要对 $19960515$ 取模。

---

**代码**：
```cpp
#include &lt;iostream&gt;
#include &lt;cstring&gt;
#include &lt;cstdio&gt;
#include &lt;algorithm&gt;
#include &lt;cmath&gt;
#include &lt;sstream&gt;
#include &lt;vector&gt;
#include &lt;queue&gt;
#include &lt;stack&gt;
#include &lt;map&gt;
#include &lt;set&gt;
#include &lt;unordered_map&gt;
#include &lt;unordered_set&gt;

using namespace std;

#define IOS ios::sync_with_stdio(false), cin.tie(nullptr), cout.tie(nullptr)
#define re register
#define fi first
#define se second
#define endl '\n'

typedef long long LL;
typedef pair&lt;int, int&gt; PII;
typedef pair&lt;LL, LL&gt; PLL;

const int MOD = 19960515;
const int MAX_PRE = 90; // 预处理长度，斐波那契数列第90项已远超1e18

LL fib[MAX_PRE]; // 存储原始斐波那契数列，fib[0]=0, fib[1]=1, fib[2]=1, ...

void init() {
    fib[0] = 0;
    fib[1] = 1;
    for (int i = 2; i &lt; MAX_PRE; i++) {
        fib[i] = fib[i - 1] + fib[i - 2];
        // 注意：此处不取模，保留原始值用于计算t
    }
}

void solve() {
    LL i, g, j;
    cin &gt;> i >> g >> j;

    // 检查i和j是否在预处理范围内，并且x_i (fib[i]) 不为0
    if (i >= MAX_PRE || i &lt; 1 || fib[i] == 0 || j &gt;= MAX_PRE || j &lt; 1) {
        cout &lt;&lt; -1 &lt;&lt; endl;
        return;
    }

    LL x_i = fib[i];
    LL y_i = fib[i - 1]; // 根据规律，y_i = x_{i-1}

    // 检查(g - y_i)是否能被x_i整除，并且得到的t为正整数
    if ((g - y_i) % x_i != 0) {
        cout &lt;&lt; -1 &lt;&lt; endl;
        return;
    }
    LL t = (g - y_i) / x_i;
    if (t &lt;= 0) {
        cout &lt;&lt; -1 &lt;&lt; endl;
        return;
    }

    // 计算G_j = x_j * t + y_j，并对MOD取模
    LL x_j = fib[j];
    LL y_j = fib[j - 1];
    // 先对乘法部分取模，防止溢出（虽然t和x_j可能很大，但取模后计算）
    LL res = ((x_j % MOD) * (t % MOD)) % MOD;
    res = (res + (y_j % MOD)) % MOD;

    cout &lt;&lt; res &lt;&lt; endl;
}

int main() {
    IOS;

    init(); // 预处理原始斐波那契数列

    int _ = 1;
    cin &gt;> _;

    while (_--) {
        solve();
    }

    return 0;
}
```
