**思想**：
* 思路：
  * 青蛙一共跳了 `k` 次，则：
    * 当 `k` 为奇数时，向右边跳了 `k / 2 + 1` 次，向左边跳了 `k / 2` 次。
    * 当 `k` 为偶数时，向右边跳了 `k / 2` 次，向左边跳了 `k / 2` 次。

**代码**：
```cpp
#include &lt;bits/stdc++.h&gt;
using namespace std;

typedef long long LL;

void solve(){
    LL a, b, k; cin >> a >> b >> k;
    cout &lt;&lt; a * (k / 2 + (k % 2LL != 0)) - b * (k / 2) &lt;&lt; endl;
}

int main(){
    int _; cin &gt;> _;
    while(_ --) solve();
    return 0;
}
```
