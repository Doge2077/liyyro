# 比例简化


[原题链接](https://www.acwing.com/problem/content/description/460/)

**思路1**：

* 暴力枚举。
    * 枚举分子 `i` 和分母 `j`，利用 `eps` 记录当前最小差值来判断更新条件。

**代码**：
```cpp
#include <bits/stdc++.h>
using namespace std;

void solve(){
    double a, b; int L; 
    cin >> a >> b >> L;
    int aa, bb; double eps = 1e6; 
    for(int i = 1; i <= L; i ++){
        for(int j = 1; j <= L; j ++){
            double n = (double)i / j, m = a / b;
            if(n >= m && n - m < eps) aa = i, bb = j, eps = n - m;
        }
    }
    cout << aa << ' ' << bb << endl;
}

int main(){
    solve();
    return 0;
}
```

**思路2**：

* 二分。
    * 枚举分子 `i`，二分找到分母 `j` 优化掉一层循环。

```cpp
#include <bits/stdc++.h>
using namespace std;

void solve(){
    double a, b; int L; 
    cin >> a >> b >> L;
    int aa, bb; double eps = 1e6;
    for(int i = 1; i <= L; i ++){
        int l = 1, r = L;
        double m = a / b;
        while(l < r){
            int mid = l + r + 1 >> 1;
            double n = (double)i / mid;
            if(n >= m) l = mid;  //满足条件则更新左边界
            else r = mid - 1;    //反之更新右边界
        }
        double n = (double)i / l;
        if(n >= m && n - m < eps) aa = i, bb = l, eps = n - m;
    }
    cout << aa << ' ' << bb << endl;
}

int main(){
    solve();
    return 0;
}
```
