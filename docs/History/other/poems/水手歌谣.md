# 水手歌谣


呦——呵！  
篝火之夜  

呦——呵！  
载歌载舞  

呦——呵！  
月明之夜  

呦——呵！  
吐出碎骨  

呦——呵！  
暴雷之夜  

呦——呵！  
升起船锚  

呦——呵！  
海啸之夜  

呦——呵！  
扬起船帆  

呦——呵！  
以星星为指引  

呦——呵！  
向着宝藏定航  

呦——呵！  
把鱼钩挂满船舷，  
让旗飘扬；  
把朗姆酒瓶喝光，  
让桨燃烧；  
把谦逊送给暴雨，  
让舵猖狂；  
把死亡交给勇气，  
让爱破碎！  

呦——呵！  
拒绝忍耐，  
拒绝满足；  

呦——呵！  
不屈服于海怪，  
不屈服于巨浪；  

呦——呵！  
大雾之夜  

呦——呵！  
回到船舱  

呦——呵！  
鱼腥之夜  

呦——呵！  
丢掉宝藏  

呦——呵！  
归途之夜  

呦——呵！  
星月不熄  

呦——呵！  
水手之夜  

呦——呵！  
把酒寻欢  

呦——呵！
