# 致母亲


母亲，我能为您做些什么？

母亲，我还能爱您多久？

爱，一万年太久，太久

那些花儿在您的脚下成熟，

可惜白发太长，时间太短

我们时间的交集，太短

爱一万年的时间，不够

搬过的一袋袋化肥，一百斤

扛过的一袋袋盐，一百八十斤

是什么撑得起您壮汉般的身体

又是什么压垮脊柱落下了隐疾

爱，一万年太久，太久

会有哭着的人出生，也会

有人安静离去，在这之间

母亲，您的爱也在那之后

爱一万年的时间，只有您能做到

我做您儿子懂事太晚，请您原谅

2024.5.21 后记：母亲节买的耳钉现在才给到母亲手里，她却舍不得戴，这几天在给外孙女做被褥一类的东西，兴奋使她忘记了疲惫，致使旧疾复发，给她按摩完毕立马又下单了几款按摩仪，希望能有帮助。每次回家，都能看到母亲白发增多，愈发切身体会到了她在变老，我做儿子的，还能赡养母亲多久呢。祝愿母亲身体健康，祝愿全天下的母亲幸福长寿。
