# 诸事不顺-行有不得反求诸己-《孟子》


月光亲吻河水，惊醒了水底的新娘。这四月的新娘，浮出水面，和远处大喜的村庄。

篝火摇动，少年少女的影子，鼓点，歌声，脚步。黄铜的大锅滚沸，酒器叮当。

四月的新娘，你那昏沉的爱人，垂挂在（早已呕吐和烂醉的）村庄里情人的肩头。

树叶沙沙，你的眼角沙沙。二十根骨头深深嵌入，不可撼动的河床，脸也埋下。

四月的新娘，你不需要鸟巢和胸膛，我们自己便是。

四月，你驮着我像一口棺材，在河水中漂浮，驮着我，和今夜疲惫的月光。

2022.4.1
